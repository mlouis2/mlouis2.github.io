Skeleton Attack!

Fall 2017

Technologies: JavaScript, HTML, CSS

    For my introduction to programming course, we were tasked with creating a "chaser" game. Mine features a human sprite being chased around by skeletons, with several different power-ups to help keep you alive. The player's high score is stored in local browser storage so that it doesn't go away immediately, and if you enter the Konami code into the game, you will see a special surprise!

Check it out on GitHub and the live page

LMUgrowl

September 2017 - Present

Technologies: Python, HTML, CSS, Google App Engine

    At my university, I noticed lots of students spending time calculating how much they could spend per day with their remaining meal plan dollars. It is a pain to calculate, and it seemed there was no better way. I created LMUgrowl with the intent of doing all of the hard work for students who aren't sure whether or not they are on-track with their meal plan spending.

Check it out on GitHub and the live page

Movie Night

Summer 2017

Technologies: Python, JavaScript, HTML, CSS, Google App Engine

    Movie Night was created by two other students and I as our Google CSSIx final project. It is intended to help people determine what movie they should watch by asking questions about genre, rating, and more. It utilizes "The Movie Database" API, Google Login Authentication, and a database to remember a user's selected movie.

Check it out on GitHub and the live page
